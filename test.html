Тест системы компонентов

Эта страница демонстрирует работу системы компонентов

Как это работает:

  • Компонент футера загружается автоматически через JavaScript
  • Код футера написан один раз в components/footer.html
  • На каждой странице только одна строка: <div data-component="footer"></div>
  • При изменении компонента обновляются все страницы автоматически
